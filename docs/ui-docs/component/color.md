# 色彩 Color
根据系统沉淀以及业务的多元场景，探索和定义出一套特定的色彩通用方案，帮助所搭建的产品提供一致的色彩感受。

## 默认主色/品牌深色
主色是奠定产品基调、传递产品性格、体现产品价值的核心元素，使用频率高，常用于强调信息以及引导操作。UED/Web规范选取蓝色作为默认主色
